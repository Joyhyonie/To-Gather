* 프로젝트 상세 소개



✔ 프로젝트 소개 입력 방식은 아래 두 가지입니다.
  - 블로그형 (이미지-글-이미지-글 형태)
  - 웹페이지형 (하나의 이미지 형태, 자체 제작)
✔ 문단과 문단 사이 간격이 넓습니다. 간격을 줄이시려면 shift + enter 를 사용해주세요.
✔ 문장에 적용한 효과를 삭제하려면 해당 문장을 선택한 뒤 적용한 효과 버튼을 한번 더 눌러주세요.
✔ 타 파일에서 문장을 복사, 붙여넣기 전에는 메모장 등에 한번 거쳐서 입력해주세요.
✔ 최대 노출 가능한 이미지 가로 사이즈는 600px 입니다.
✔ 권장 비율을 지키지 않았을 경우 이미지 일부가 잘린 채 노출되는 등 다르게 보일 수 있습니다.
✔ 이미지 삽입을 통해 GIF 파일 입력이 가능합니다

Supporter에게 지급하실 리워드를 등록해주세요 :)
